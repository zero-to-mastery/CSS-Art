Solar System

I know this is not the first one on Animation-Nation, but I'm a space and physics lover, and I wanted to create mine :)

Here is a true time scaled solar-system, which means that every objects have a time relative to an Earth year. Here 1 year = 30 seconds.

Note the css tricks for the Saturn rings (box-shadow) and reverse animation to compensate the orbit.
